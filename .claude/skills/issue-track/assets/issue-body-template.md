### 示例

> **OIDC 登录认证（A-01）**
>
> 1. 用户访问平台，未登录状态下重定向到 OIDC Provider（含 PKCE）
> 2. 用户在企业 SSO 页面输入凭据完成认证
> 3. OIDC Provider 回调平台，平台用 authorization code 换取 token
> 4. 平台从 token 中解析用户信息，首次登录自动创建 User 记录（默认 viewer 角色）
> 5. 平台生成 session 返回前端，前端存储 token 并跳转到主页
> 6. 用户后续请求携带 token，平台校验有效性
> 7. token 过期时前端用 refresh token 自动刷新
> 8. 用户点击登出，平台调用 OIDC RP-Initiated Logout 结束会话
